# Notes App

A simple notes app to create update read delete notes in markdown format

# Set up

- `git clone` (repo url)
- `cd Notes-App`
- `npm i` (install node modules)
- `npm run dev` (to run the app locally)

# Tech Stack

- React
- TypeScript
